# Convert Dollar


## Início

Conversor de dólar para reais. Que puxa automaticamente a API do Banco Central, atualizando o valor do dólar automaticamente.


### Pré-requisitos:

Você precisa do NodeJS e do NPM instalado em sua máquina.

```
npm install
npm run dev
```

## Layout:
